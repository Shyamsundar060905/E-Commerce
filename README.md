# E-Commerce Web Application

This is a full-stack e-commerce web application built using the MERN stack 
(React, Express, MongoDB, Node.js). The application allows users to browse 
products, add items to a cart, and manage user authentication.

The project demonstrates full-stack development including REST API development, 
database management, authentication, and frontend UI development.

## Features
- User Authentication (Login / Signup)
- Product Listing
- Shopping Cart
- Order Placement
- REST API Integration
